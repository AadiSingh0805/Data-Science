# Data-Science
Documenting my data science journey through practical notebooks, projects, and experiments in Python, statistics, and machine learning.

## Projects

### AB_Testing_CTR
An A/B testing analysis focused on click-through rate (CTR). The notebook works with a 20,000-row dataset, compares control vs experimental groups, and evaluates whether the observed lift is statistically and practically significant.

The analysis includes:
- exploratory inspection of the click data
- CTR comparison between control and experimental groups
- z-test / hypothesis testing for significance
- confidence interval estimation
- practical significance check using MDE

### Superstore Sales Analysis
A sales exploration notebook built around the Superstore dataset. It covers data cleaning, customer and segment breakdowns, shipping mode analysis, state and city summaries, product/category performance, and time-based sales trends.

The analysis includes:
- sales by state visualization for the USA
- monthly sales heatmap by year
- repeat customer and customer sales analysis
- yearly and monthly sales trend charts
